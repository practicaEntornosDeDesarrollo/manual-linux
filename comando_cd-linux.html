Comando cd en Linux

El comando cd (change directory) se utiliza en sistemas Linux para cambiar de directorio dentro del sistema de archivos.

Sintaxis básica

cd [ruta]

Ejemplos comunes

1. Ir a un directorio específico

cd /home/usuario/documentos

Cambia al directorio indicado por la ruta absoluta.

2. Volver al directorio anterior

cd ..

Sube un nivel en la estructura de directorios.

3. Ir al directorio personal (home)

cd ~

Te lleva directamente a tu carpeta personal.

4. Volver al último directorio visitado

cd -

Alterna entre el directorio actual y el anterior.

Rutas absolutas vs relativas

  • Ruta absoluta: comienza desde la raíz /.
  • Ruta relativa: depende del directorio actual.

Consejos útiles

  • Usa Tab para autocompletar rutas.
  • Combina cd con ls para explorar directorios.

Conclusión

El comando cd es esencial para navegar por el sistema de archivos en Linux de forma eficiente.
