Comando rm en Linux

El comando rm (remove) se utiliza para eliminar archivos y directorios en sistemas Linux.

Atención: Los archivos eliminados con rm no se pueden recuperar fácilmente.

Sintaxis básica

rm [opciones] archivo

Ejemplos comunes

1. Eliminar un archivo

rm archivo.txt

Elimina el archivo especificado.

2. Eliminar múltiples archivos

rm archivo1.txt archivo2.txt

3. Eliminar un directorio vacío

rm -d carpeta

4. Eliminar directorios y su contenido

rm -r carpeta

Elimina la carpeta y todo lo que contiene.

5. Forzar eliminación sin preguntar

rm -f archivo.txt

6. Eliminación recursiva y forzada

rm -rf carpeta

Muy potente y peligroso: elimina todo sin confirmación.

Opciones más usadas

  • -r o -R: elimina directorios recursivamente.
  • -f: fuerza la eliminación sin pedir confirmación.
  • -i: pide confirmación antes de borrar.

Consejos útiles

  • Usa rm -i para evitar borrar archivos por error.
  • Ten mucho cuidado con rm -rf, especialmente como administrador.
  • Comprueba siempre la ruta antes de ejecutar el comando.

Conclusión

El comando rm es muy útil pero debe usarse con precaución, ya que elimina archivos de forma permanente.
