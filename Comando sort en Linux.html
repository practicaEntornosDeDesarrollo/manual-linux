Comando sort en Linux

¿Qué es sort?

El comando sort sirve para ordenar líneas de texto, ya sea de un archivo o de la salida de otro comando.

¿Qué hace exactamente?

sort organiza líneas automáticamente:

  • 🔤 Alfabéticamente (por defecto)
  • 🔢 Numéricamente
  • 📊 Por columnas o campos

Uso básico

sort archivo.txt

Muestra el contenido ordenado alfabéticamente.

Ejemplos prácticos

Orden alfabético

sort nombres.txt

Orden inverso

sort -r nombres.txt

Orden numérico

sort -n numeros.txt

Eliminar duplicados

sort -u lista.txt

Ordenar por columna

sort -k 2 datos.txt

Separador personalizado

sort -t "," -k 2 datos.csv

Guardar resultado

sort archivo.txt > ordenado.txt

Uso con otros comandos

cat archivo.txt | sort ps aux | sort -k 3

Permite ordenar resultados de otros comandos.

Resumen

  • 🔤 Ordena texto
  • 🔢 Soporta números (-n)
  • 🔁 Orden inverso (-r)
  • 🧹 Elimina duplicados (-u)
  • 📊 Ordena por columnas (-k)
