Comando chmod en Linux

¿Qué es chmod?

El comando chmod sirve para cambiar los permisos de archivos y directorios, controlando quién puede leer, escribir o ejecutar un archivo.

¿Qué hace exactamente?

Permite definir permisos para:

  • 👤 Usuario (u)
  • 👥 Grupo (g)
  • 🌍 Otros (o)

Tipos de permisos:

  • 📖 r (leer)
  • ✏️ w (escribir)
  • ⚙️ x (ejecutar)

Uso básico

chmod [permisos] archivo

Método numérico

Cada permiso tiene un valor:

  • r = 4
  • w = 2
  • x = 1
Número Permiso
7 rwx
6 rw-
5 r-x
4 r--
chmod 755 archivo.sh

Ejemplos prácticos

Permisos solo para el dueño

chmod 700 archivo.txt

Solo lectura

chmod 444 archivo.txt

Hacer ejecutable

chmod +x script.sh

Quitar escritura

chmod -w archivo.txt

Permisos específicos

chmod u+x archivo.sh

Recursivo

chmod -R 755 carpeta/

Método simbólico

chmod u+rwx,g+rx,o+rx archivo

Importante

  • ⚠️ Permisos incorrectos pueden causar problemas
  • ⚠️ 777 da acceso total a todos (no recomendado)

Resumen

  • 🔐 Controla permisos
  • 👤 u / 👥 g / 🌍 o
  • 📖 r | ✏️ w | ⚙️ x
  • 🔢 Método numérico
  • 🔤 Método simbólico
