Comando cut en Linux

¿Qué es cut?

El comando cut sirve para extraer partes específicas de cada línea de un archivo o texto. Es muy útil con datos estructurados como CSV o logs.

¿Qué hace exactamente?

cut permite cortar texto según:

  • 📍 Posición de caracteres
  • 📊 Columnas (campos)
  • 🔤 Delimitadores (coma, espacio, etc.)

Uso básico

cut [opciones] archivo.txt

Ejemplos prácticos

Cortar por caracteres

cut -c 1-5 archivo.txt

Un carácter específico

cut -c 3 archivo.txt

Primera columna (CSV)

cut -d "," -f 1 archivo.csv

Varias columnas

cut -d "," -f 1,3 archivo.csv

Rango de columnas

cut -d ":" -f 1-3 /etc/passwd

Con otros comandos

ps aux | cut -c 1-10 ls -l | cut -d " " -f 1

Importante

  • -d → separador
  • -f → campos (columnas)
  • -c → caracteres

Resumen

  • ✂️ Extrae partes de texto
  • 📊 Ideal para columnas
  • 🔤 Funciona por posición o separador
  • ⚡ Muy útil con tuberías
