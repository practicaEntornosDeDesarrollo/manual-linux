Comando grep en Linux

¿Qué es grep?

El comando grep sirve para buscar texto dentro de archivos o salidas de otros comandos. Su nombre viene de Global Regular Expression Print, ya que usa expresiones regulares para encontrar patrones.

¿Qué hace exactamente?

grep lee línea por línea y muestra solo aquellas que coinciden con un patrón que tú indiques.

Uso básico

grep "texto" archivo.txt

Muestra todas las líneas que contienen "texto".

Ejemplos prácticos

Buscar una palabra

grep "error" log.txt

Ignorar mayúsculas/minúsculas

grep -i "error" log.txt

Mostrar número de línea

grep -n "error" log.txt

Buscar en varios archivos

grep "main" *.c

Buscar en carpetas

grep -r "config" .

Invertir búsqueda

grep -v "error" log.txt

Contar coincidencias

grep -c "error" log.txt

Uso con otros comandos

ps aux | grep firefox

Busca procesos relacionados con Firefox.

Resumen

  • 🔍 Busca texto en archivos
  • 🧩 Usa patrones (expresiones regulares)
  • ⚡ Muy potente con otros comandos
  • 📂 Funciona con múltiples archivos
